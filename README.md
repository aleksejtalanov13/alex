# Как собрать?

Для сборки берем Android Studio и открываем там проект
Игнорируем предложение обновить всякие плагины и продолжаем
Теперь нам нужно запустить приложение
Подготовьте ваш телефон с включенной отладкой по USB и включенной установкой через USB

Следуйте этим шагам
1. Дождитесь подготовки среды Android Studio
2. Подключите телефон
3. Разрешите отладку
4. Выберите устройство и нажмите Run app
5. Дождитесь завершения сборки
6. Если телефон спросит разрешение на установку приложения смело разрешайте
7. После установки приложение запустится
8. Пользуйтесь!
